# 绯羽国际学校学生手册

## 第一章：总则

1.  **欢迎与使命**
    - 欢迎词与对学生的期望。
    - 学校使命、愿景与核心价值观阐述。
2.  **手册宗旨**
    - 本手册旨在明确学生在校期间的权利、责任与行为规范，是学生在校学习、生活的根本指南。
3.  **适用范围**
    - 本手册适用于所有在绯羽国际学校注册的全日制与非全日制学生。
4.  **修订与解释权**
    - 学校有权根据实际情况修订本手册，修订内容将正式通知。本手册的最终解释权归绯羽国际学校所有。

---

## 第二章：学籍与学术管理

1.  **入学与注册**
    - 入学资格、录取流程及每学年的注册报到规定。
2.  **学业要求**
    - 学分制度、课程选修与考核方式。
    - 考勤制度（请假流程、缺勤与旷课的处理）。
3.  **学术诚信政策（核心章节）**
    - 学术诚信的定义与重要性。
    - 抄袭、剽窃、作弊等学术不端行为的明确定义与示例。
    - 对学术不端行为的调查程序与分级处罚措施。
4.  **成绩与申诉**
    - 成绩评定、查询与归档。
    - 学生对成绩或学术处分提出申诉的正式流程。

---

## 第三章：学生行为规范

1.  **基本原则**
    - 尊重、责任、诚信、包容的行为准则。
2.  **校园行为规范**
    - 课堂纪律、图书馆、实验室等公共场所的行为要求。
3.  **网络与电子设备使用规范**
    - 学校网络资源的使用责任，禁止网络霸凌、散播不当信息。
    - 课堂上电子设备的使用规定。
4.  **着装要求**
    - 在校期间及特定活动（如典礼、晚会）的着装标准。
5.  **反歧视与反骚扰**
    - 明确禁止任何形式的歧视、霸凌与骚扰，并提供举报渠道。

---

## 第四章：校园生活与服务

1.  **一卡通管理**
    - 功能：身份凭证、门禁钥匙、电子钱包。
    - 使用规则：仅限本人使用，妥善保管，避免损坏。
    - 挂失与补办：遗失后立即前往**学生事务中心**或使用自助终端挂失的具体流程。
    - 有效期与销户：毕业或离校时需办理销户并退还余额。
2.  **宿舍管理**
    - 住宿规定、访客制度、安全检查与作息时间。
3.  **学生组织与活动**
    - 学生社团的申请、管理与活动开展指南。
4.  **支持服务**
    - **学生事务中心**：提供日常事务咨询与服务。
    - 心理咨询中心、学业指导中心、职业发展中心的服务介绍。

---

## 第五章：校园设施与安全

1.  **设施使用**
    - 教学楼、图书馆、体育场馆、餐厅等设施的开放时间与使用规则。
2.  **安全守则**
    - 紧急情况（如火警、地震）的疏散流程与集合地点。
    - 门禁管理：**“受限区域，授权进入”** 原则，必须随身携带一卡通。
3.  **资产保护**
    - 学校财产（包括一卡通）为**学校受控资产**，应爱惜使用。拾获他人卡证应主动交还至**网络安全部**。
4.  **网络安全**
    - 遵守国家法律法规和学校网络管理规定，由**校园网络安全部**负责监督与执行。

---

## 第六章：奖励与处分

1.  **奖励制度**
    - 对学业优异、领导力、社会服务等方面表现突出的学生的表彰与奖励。
2.  **纪律处分**
    - 根据违规行为的严重程度，处分为多个等级（如警告、严重警告、记过、留校察看、开除学籍）。
