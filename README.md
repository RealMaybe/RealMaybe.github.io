## 关于构建

项目采用现代前端技术栈开发，确保了极致的开发体验与流畅的用户体验。

包括但不限于：

- Vite
- Vue 3
- TypeScript
- Pinia
- Less
- Font Awesome
- Git

整个架构轻快而灵活，正如我所期待的个人空间一样，能够自由地生长和演化。
